## 🚀 日常操作

### 添加新工具（3步）

```
1. 打开 tool-editor.html → 填写信息 → 生成 JSON
2. 复制代码到 src/data/tools.json
3. 运行：git add . → git commit -m "添加XXX" → git push
```

### 修改工具信息

```
1. 编辑 src/data/tools.json 或 toolDetails.json
2. 运行：git add . → git commit -m "更新XXX" → git push
```

## 🔧 常用命令

### 本地开发

```bash
npm run dev          # 启动开发服务器
npm run build        # 构建生产版本
```

### Git 操作

```bash
git status           # 查看状态
git add .            # 添加所有修改
git commit -m "说明" # 提交
git push             # 推送到 GitHub
git log --oneline    # 查看历史
```

### 回滚操作

```bash
git reset --hard HEAD~1    # 回滚到上一版本
git push --force           # 强制推送
git reset --hard v1.0      # 回滚到 v1.0
```

## 🆘 故障处理

### 网站打不开

1. 检查 Vercel 状态
2. 检查域名解析：`ping catai.cc.cd`
3. 查看部署日志

### 页面显示异常

1. 验证 JSON 格式：https://jsonlint.com
2. 本地测试：`npm run build`
3. 清除缓存：Ctrl + F5

### Logo 不显示

1. 手动上传 Logo 到 `public/logos/`
2. 修改 tools.json 中的 logo 路径
